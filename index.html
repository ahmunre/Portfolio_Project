Marvel Quiz

This Application tests the trivia knowledge of both the casual and avid Marvel Cinematic Universe fan. There are ten questions covering a range of questions, with a platform friendly user interface and end-user feedback once the quiz is ended. The application contains a page for when the end-user responses are correct complete with feedback, a page for when the answer is incorrect also with the appropriate feedback and finally a page to fully restart the application.
